Limbajul html

HTML (HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html îl găsiți în imaginea următoare care se află la adresa https://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png
Imaginea are atributul height de valoare 147.

Tag-urile html cele mai utilizate în realizarea tabelelor sunt:

table definește un table
tr un rând dintr-un tabel
td o celulă dintr-un tabel

Observatie: Pentru a pune un contur unui tabel se poate (dar nu se recomanda) să se pună atributul border="1".
La validare va da eroare: The border attribute on the table is absolute

EX 3

Pentru a crea liste se pot folosi tag-urile:

  1. ul - listă neordonată
  2. ol - listă ordonată
  3. li - item din listă
  4. dl - listă cu descrieri
  5. dt - termen dintr-o listă cu descrieri
  6. dd - descrierea dintr-o listă cu descrieri

Pentru a insera un document HTML în altul se poate folosi tag-ul < iframe >
Astfel, exemplul de mai jos inserează conținutul de la adresa https://html.spec.whatwg.org/multipage/
